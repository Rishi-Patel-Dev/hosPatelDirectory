# hosPatelDirectory
This project is a menu-driven Java console application designed to manage a hospitalâ€™s staff and patient directories. It provides an easy-to-use text-based interface for adding, searching, sorting, assigning, and managing records. The system supports both staff and patient data and includes import/export functionality for persistent storage.

# Features
  - Staff Management
  - Add new staff members
  - Assign staff to specific patients
  - Search for staff by ID
  - Display all staff records
  - Remove staff members
  - Sort staff by name
  - Import and export staff data to/from files

# Patient Management

  - Add new patients
  - Assign patients to staff
  - Search for patients by ID
  - Update patient health status
  - Display all patient records
  - Remove patients
  - Sort patients by name
  - Import and export patient data

# Program Flow

The application runs through a structured menu system:

  - Main Menu
  - Staff Directory
  - Patient Directory
  - Exit
  - Staff Menu
  - Patient Menu

Each section supports operations such as adding, viewing, modifying, assigning, sorting, and file I/O.

# Technologies Used

  - Java (Console-based)
  - Object-Oriented Programming
  - File Handling (Import/Export)
  - Collections Framework
  - Input Validation & Error Handling

# Purpose

This application simulates a lightweight Hospital Information System (HIS) suitable for academic projects, practicing Java fundamentals, or demonstrating file handling and OOP design patterns.
